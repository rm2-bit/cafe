# ☕ Cafe 365 - Artisanal Coffee Experience

**Cafe 365** is a modern, responsive landing page for a premium coffee roastery and café. Designed with a sophisticated "Roasted Espresso" aesthetic, this project showcases a decade of coffee expertise through a clean, interactive user interface.

## 🌟 Features

- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
- **Dynamic Product Showcase:** A clean grid layout featuring signature brews like Espresso, Cappuccino, and Latte.
- **Interactive Search:** Find your favorite brew instantly with an integrated search bar that scrolls to your choice.
- **Modern UI/UX:** Smooth scrolling, hover effects, and a rich, coffee-inspired color palette.
- **Secure Access:** Integrated Login and Signup routing for a community-focused shopping experience.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom variables, Flexbox, and Grid layouts for a modern look.
- **JavaScript (ES6):** Dynamic DOM manipulation for search, scrolling, and mobile navigation.
- **Icons:** Powered by [FontAwesome](https://fontawesome.com/) and [BoxIcons](https://boxicons.com/).
